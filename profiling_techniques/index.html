    Introduction to Profiling CPU Cycles¶

    • Profiling is essential for understanding how much CPU time different pieces of code consume.

    • We will use Python's profiling tools such as cProfile, timeit, and line_profiler to compare the performance of multiple-if statements and switch-case patterns (using dictionary-based switch implementations in Python).

    • Profiling will allow us to compare the CPU cycles consumed by these conditional structures, providing theoretical insights with practical evidence.

    CPU Cycles in Multiple-If Statements¶

    • Multiple-if statements are simple but can result in higher CPU cycles when there are many conditions to evaluate.

    • Here’s an example of profiling a multiple-if block using cProfile:

    import timeit
import cProfile

def multiple_if(x):
    if x == 1:
        return 'one'
    elif x == 2:
        return 'two'
    elif x == 3:
        return 'three'
    elif x == 4:
        return 'four'
    elif x == 5:
        return 'five'
    elif x == 6:
        return 'six'
    else:
        return 'unknown'

# Profiling with cProfile
cProfile.run('multiple_if(6)')

# Measuring execution time with timeit
time_multiple_if = timeit.timeit('multiple_if(6)', globals=globals(), number=1000000)
print(f"Execution time for multiple-if: {time_multiple_if}")

    def switch_case(x):
    cases = {
        1: 'one',
        2: 'two',
        3: 'three',
        4: 'four',
        5: 'five',
        6: 'six'
    }
    return cases.get(x, 'unknown')

# Profiling with cProfile
cProfile.run('switch_case(6)')

# Measuring execution time with timeit
time_switch_case = timeit.timeit('switch_case(6)', globals=globals(), number=1000000)
print(f"Execution time for switch-case: {time_switch_case}")

    import matplotlib.pyplot as plt

# Ensure these are numeric values (execution times)
time_multiple_if = 0.085  # Example time for multiple-if (in seconds)
time_switch_case = 0.042  # Example time for switch-case (in seconds)

# Data for plotting
conditions = ['Multiple-If', 'Switch-Case']
times = [time_multiple_if, time_switch_case]  # This should be a list of numbers

# Plotting the bar chart
plt.bar(conditions, times, color=['blue', 'green'])
plt.xlabel('Method')
plt.ylabel('Execution Time (seconds)')
plt.title('CPU Cycle Comparison: Multiple-If vs Switch-Case with More Than 5 Conditions')
plt.show()

    import time

start = time.perf_counter()
multiple_if(3)
end = time.perf_counter()
print(f"Multiple-If CPU Time: {end - start} seconds")

start = time.perf_counter()
switch_case(3)
end = time.perf_counter()
print(f"Switch-Case CPU Time: {end - start} seconds")
